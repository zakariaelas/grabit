# Installation

Use git to clone the grabit repository locally

with HTTPS:

```
git clone git@github.com:zakariaelas/grabit.git
```

with SSH:

```
git clone git@github.com:zakariaelas/grabit.git
```

Once you change directories to `backend` or `frontend`, you will need to install all dependencies.

```
cd backend
npm install
cd ../frontend
npm install
```
